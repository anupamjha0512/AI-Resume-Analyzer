# AI Resume Analyzer

A web-based Resume Analyzer that extracts skills from PDF resumes, calculates an ATS score, identifies missing skills, and provides personalized recommendations.

## Live Demo

https://ai-resume-analyzer-6elt.onrender.com

## Features

* PDF Resume Upload
* ATS Score Calculation
* Skill Detection
* Missing Skills Analysis
* Personalized Recommendations
* Responsive UI

## Tech Stack

* Python
* Flask
* HTML
* CSS
* PyPDF2
* Render

## Run Locally

### 1. Install dependencies

pip install -r requirements.txt

### 2. Start the application

python app.py

### 3. Open browser

http://127.0.0.1:5000
